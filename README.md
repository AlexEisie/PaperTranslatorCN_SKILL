# 告诉GPT如何进行一次合格的论文翻译  
### Tell your GPT how to carry out a proper thesis translation  
  
## 怎么用  
###  How To Use
将paper-pdf-to-cn-latex文件夹放置在<USERHOME>\.codex\skills\.system下以自动加载  
合适的路径安排如下：  
skills\.system\paper-pdf-to-cn-latex\SKILL.md   

Put paper-pdf-to-cn-latex folder in <USERHOME>\.codex\skills\.system so codex can use it
It should be like: 
skills\.system\paper-pdf-to-cn-latex\SKILL.md   

## 注意事项  
## Attention 
使用前安装XeLaTex环境以用于PDF编译渲染，或者将LaTex项目转移到OverLeaf编译
Ensure XeLaTex is available on your system or compile the project on OverLeaf
